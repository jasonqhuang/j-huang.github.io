Car Color Identifier Neural Network

Python, Pandas, Numpy, Convolutional Neural Network, Keras, Tensorflow

This project was my capstone for General Assembly Data Science Immersive. The capstone was a Convolutional Neural Network that can predict the color of a car given any image of one. The majority of the code is broken up into three parts:

  • Identifying the color of the car in a photo and retrieving the RGB values.
  • Classifying images via RGB color for model training.
  • Training and validating the Convolutional Neural Network (CNN) using Tensorflow w/ Keras.
All the code was written in python. The CNN model architecture was adapted from previous work by Reza Fuad Tachmadi and I Ketut Eddy Purnama in their paper Vehicle Color Recognition using Convolutional Neural Network.

PyDisaster

Python, Pandas, Flask, Google Maps API, PIL, Geolocation, HTML

PyDisaster was developed as a locally hosted app that would input disaster photos and compile a database of geolocations during early stage disatster relief efforts. PyDisaster can retrieve geolocations (latitude and longitude) via EXIF data from both Iphone and Android phone photos and plot it using Google Map's API. The app was locally hosted via Flask and is relatively low resource intensive which is critical during disaster time.
